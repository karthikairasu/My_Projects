Identifiers and Keywords

Identifier Naming Rules

  • First character should be an alphabet, underscore _ symbol or dollar $ symbol
  • Other characters may be alphanumerical, underscore or dollar symbols
  • Should not use keywords and reserved words as Identifier

JavaScript Syntax

  • Case sensitive language
  • Using semicolons to terminate statements is good practice
