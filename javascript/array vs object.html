Array & Object

Object : represent a special data type that is mutable and can be used to store a collection of data (rather than just a single value)
=> Unordered sets of name-value pair

            const obj = {
                name: 'Prashant Yadav',
                age: '26',
                gender: 'Male',
                getDetails: function(){
                return `${this.name} is ${this.age} years old`;
                }
            };
            
            console.log(obj.name);
            //"Prashant Yadav"
            
            console.log(obj.age);
            //"26"
            
            console.log(obj.getDetails());
            //"Prashant Yadav is 26 years old"

Array : Arrays are a special type of variable that is also mutable and can also be used to store a list of values
